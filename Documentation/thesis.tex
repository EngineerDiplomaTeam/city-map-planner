\nabstract{
	Celem niniejszej pracy było stworzenie progresywnej aplikacji internetowej pozwalającej na kompleksowe tworzenie planu zwiedzania atrakcji turystycznych w Gdańsku uwzględniającego dane czasu rzeczywistego, takie jak godziny otwarcia, komunikacja miejska i prognoza pogody.
	W ramach pracy stworzono system składający się z trzech komponentów: frontendu, \gls{backend}u oraz bazy danych.
	Progresywna aplikacja internetowa została wykonana przy pomocy frameworka Angular w języku TypeScript oraz biblioteki RxJs.
    Serwis backendowy został stworzony w środowisku .NET w języku C\# oraz bibliotek Asp.NET.
    Baza danych to PostgreSql wzbogacony o rozszerzenia TimescaleDB.\newline
    Komunikacja między komponentami odbywała się zgodnie ze standardem REST oraz reverse proxy (Ngnix).
    Projekt został zrealizowany w podejściu ewolucyjno-przyrostowym~z~elementami Kanban.
}
